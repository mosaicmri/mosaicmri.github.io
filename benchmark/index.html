MosaicMRI Benchmark

The benchmark evaluates machine learning methods for accelerated MRI reconstruction using hidden reference targets. Every challenge uses the same secure upload and server-side evaluation pipeline.

Accelerated Reconstruction Challenge 1 Anatomy Generalization Challenge 2 Contrast Generalization Challenge 3

Problem Description

This track evaluates accelerated reconstruction at 8× acceleration. Methods are ranked by PSNR, SSIM, and NMSE on hidden test targets, with optional anatomy-specific analysis. Use multicoil_train/ and multicoil_val/ for model development, and submit predictions for multicoil_test.

Submission and Evaluation Workflow

Maximum upload size: 8 GB per submission ZIP.

  1. Use your approved dataset access email. Data Format and Quickstart
  2. Train a reconstruction model on released training/validation data.
  3. Run inference for the 8× track and keep required file naming.
  4. Package predictions into one ZIP and upload below.
  5. Track status (queued → evaluating → completed) and monitor leaderboard ranking.

Validate Before Upload

Use the challenge-specific notebook before uploading. It checks data structure, shape mismatches, naming consistency, and reports final ZIP size. 

jupyter notebook validate_challenge1.ipynb

We strongly recommend running the notebook checks before every submission. benchmark validation tools

    Problem Description

    This track studies cross-anatomy domain transfer. Models are developed without access to the held-out anatomy, and are evaluated on hidden test reconstructions from that anatomy to measure out-of-domain generalization. Submit predictions for files in anatomy_transfer_challenge/ankle/ corresponding to the ankle joint (20 files, 49.40 GiB). You may have relatively few ankle volumes, so leveraging complementary anatomies during training can improve performance.

    Submission and Evaluation Workflow

    This track evaluates held-out anatomy transfer. Maximum upload size: 500 MB per submission ZIP.

    1. Use your approved dataset access email. Data Format and Quickstart
    2. Train on released anatomies while excluding the ankle joint from training (validation remains available).
    3. Generate test reconstructions for the hidden-anatomy transfer set.
    4. Package predictions into one ZIP and upload below.
    5. Track status (queued → evaluating → completed) and compare leaderboard metrics.

      Problem Description

      This track targets contrast-domain shift and evaluates reconstruction on held-out T1-weighted fat-suppressed (T1-FS) acquisitions. Leaderboards report global and anatomy-filtered PSNR, SSIM, and NMSE. Submit predictions for files in contrast_generalization_challenge/T1_FS/. T1-FS examples can be limited (17 files, 20.74 GiB), so mixing additional anatomies/contrasts during training is often beneficial.

      Submission and Evaluation Workflow

      This track targets protocol shift in T1-weighted fat-suppressed (T1-FS) acquisitions. Maximum upload size: 700 MB per submission ZIP.

      1. Use your approved dataset access email. Data Format and Quickstart
      2. Train on released data while excluding T1-weighted fat-suppressed (T1-FS) cases from training (validation remains available).
      3. Generate reconstructions for the hidden T1-FS test split.
      4. Package predictions into one ZIP and upload below.
      5. Track status (queued → evaluating → completed) and compare leaderboard metrics.
